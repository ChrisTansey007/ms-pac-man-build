# Code of Conduct

> **This project follows the spirit of the Contributor Covenant 2.1: be respectful, be constructive, and help keep collaboration safe for everyone.**

## Our Pledge

We are committed to making participation in this project a harassment-free experience for everyone, regardless of background, identity, or experience level.

## Expected Behavior

- Be respectful and professional.
- Focus on the work, not the person.
- Give constructive feedback.
- Respect project boundaries, scope, and ownership rules.
- Assume good intent when possible.

## Unacceptable Behavior

- Harassment, threats, or abusive language.
- Discriminatory or exclusionary behavior.
- Deliberate disruption of work or review process.
- Sharing secrets, credentials, or private data without permission.

## Enforcement

Project maintainers may remove, block, or restrict contributions that violate this code of conduct.

## Reporting

If you experience or witness behavior that violates this code of conduct, report it to the project maintainers through the repository's normal support or issue workflow after removing any sensitive details.

## Scope

This code of conduct applies to all project spaces, including issues, pull requests, discussions, comments, reviews, and any other project-related communication.
