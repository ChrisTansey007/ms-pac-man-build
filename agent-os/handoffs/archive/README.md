# Archived Handoffs

This folder contains handoffs from **completed tasks**.

## Purpose

Archived handoffs provide a historical record of work done. They are useful for:
- Understanding why decisions were made.
- Auditing completed work.
- Onboarding new workers or humans to the project's history.

## What Belongs Here

- Handoffs from tasks that have moved to `done/`.
- Superseded handoffs from tasks that had multiple sessions.

## When to Archive

- When a task moves to `done/`, ALL handoffs for that task should be moved here.
- Old handoffs superseded by newer ones can be archived at any time.
