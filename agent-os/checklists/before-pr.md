# Before PR Checklist

> **Complete this checklist before opening a pull request.**

## Completion Check

- [ ] All acceptance criteria from the task file are met
- [ ] All tests pass (`npm test`)
- [ ] Linting passes (`npm run lint`)
- [ ] Validation scripts pass (`npm run validate:tasks`, `npm run validate:handoffs`)

## PR Contents

- [ ] PR description references the task ID
- [ ] PR description includes execution mode and role performed
- [ ] PR description summarizes the changes
- [ ] PR links to the handoff file
- [ ] PR links to verification evidence
- [ ] PR notes any risks or known issues
- [ ] PR notes if reassignment occurred

## Review Readiness

- [ ] Code is self-reviewed
- [ ] No obvious issues or oversights
- [ ] Ready for independent review
- [ ] PR template completed
