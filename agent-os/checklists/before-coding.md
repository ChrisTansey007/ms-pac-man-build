# Before Coding Checklist

> **Complete this checklist before writing any code.**

## Task Understanding

- [ ] Task objective is clear
- [ ] Acceptance criteria are understood
- [ ] Required capabilities confirmed
- [ ] Scope boundaries are identified
- [ ] Dependencies are resolved (or noted as blockers)

## Context Gathering

- [ ] Relevant existing code read and understood
- [ ] Related documentation reviewed
- [ ] Existing tests for affected areas reviewed
- [ ] Any design patterns or conventions noted

## Planning

- [ ] Approach planned (what files, what changes, in what order)
- [ ] Tests planned (what to test, edge cases)
- [ ] Documentation updates identified
- [ ] Risks considered

## Environment

- [ ] Development environment is set up
- [ ] Tests pass before making changes
- [ ] Linting passes before making changes
- [ ] Working on the correct branch
