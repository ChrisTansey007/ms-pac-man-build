# Reassignment Archive

> **Completed reassignment records are stored here for audit and reference.**

## Purpose

When a reassignment is complete (the new worker has successfully taken over and the task is progressing), the reassignment record is moved from the active reassignment area to this archive.

## File Naming

```
[TASK-ID]-reassignment-[YYYY-MM-DD].md
```

Example: `TASK-0005-reassignment-2026-05-06.md`

## Archive Contents

Reassignment records in this archive serve as:

- **Audit trail** — Who worked on what, when, and why it changed.
- **Pattern reference** — Common reassignment reasons can inform process improvements.
- **Continuity evidence** — Proof that the system handled worker changes correctly.
