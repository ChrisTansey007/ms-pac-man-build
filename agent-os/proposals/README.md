# Proposals

> **Structured coordination proposals for ROWS.** Use this folder when a worker needs to suggest a multi-step plan or a cross-worker coordination change without turning it into an active task yet.

## Purpose

Proposals capture ideas, sequencing, risks, and tradeoffs before work becomes a formal task.

## Files

- `proposal-template.md` — Canonical template for a proposal
- `EXAMPLE-proposal.md` — Illustrative example only

## Rules

- Proposals are not active tasks.
- Keep them short, actionable, and evidence-based.
- If a proposal is approved, promote it into `agent-os/tasks/` using the task template.
