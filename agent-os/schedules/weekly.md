# Weekly Schedule

> **Weekly cadence for the ROWS system.**

## Monday

- [ ] Review project goal — any changes needed?
- [ ] Review [`tasks/backlog/`](../tasks/backlog/) — prioritize and approve tasks for the week
- [ ] Review [`state/risk-register.json`](../state/risk-register.json) — any new risks?
- [ ] Check for stale branches that should be cleaned up

## Mid-Week (Wednesday)

- [ ] Review progress against weekly goals
- [ ] Check [`tasks/blocked/`](../tasks/blocked/) — unblock where possible
- [ ] Review open questions in [`docs/04-research/open-questions.md`](../../docs/04-research/open-questions.md)
- [ ] Run `npm run check:dod` on tasks in `review/`

## Friday

- [ ] Weekly status report generated (`npm run status:generate`)
- [ ] Archive completed handoffs
- [ ] Review decision register for new ADRs
- [ ] Plan next week's priorities
- [ ] Ensure all active workers have written handoffs
