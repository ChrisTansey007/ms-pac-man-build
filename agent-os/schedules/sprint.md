# Sprint Schedule

> **Sprint/milestone cadence.**

## Sprint Start
- [ ] Review and update [`PROJECT_GOAL.md`](../../PROJECT_GOAL.md) if needed
- [ ] Move approved tasks from `backlog/` to `ready/`
- [ ] Review [`state/dependency-map.json`](../state/dependency-map.json)
- [ ] Set sprint goal

## During Sprint
- [ ] Daily: review handoffs and worker status
- [ ] Mid-sprint: check progress, unblock tasks

## Sprint End
- [ ] All sprint tasks in `review/` or `done/`
- [ ] Run `npm run check:dod` on all sprint tasks
- [ ] Archive handoffs for completed tasks
- [ ] Update [`docs/06-release/changelog.md`](../../docs/06-release/changelog.md)
- [ ] Sprint retrospective notes
