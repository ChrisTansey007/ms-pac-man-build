# In Progress

Tasks in this folder are **actively being worked on** by a worker.

## Who Can Add Tasks Here

- The claiming worker (by moving from `claimed/`).

## What Belongs Here

- Tasks where implementation is actively happening.
- The worker's branch should exist and contain work-in-progress commits.

## How Tasks Leave Here

- **Complete:** Worker moves the task to [`../review/`](../review/) after writing handoff and evidence.
- **Blocked:** Worker moves the task to [`../blocked/`](../blocked/) if work cannot continue.

## Rules

- The worker must follow the task's scope. No expansion without escalation.
- Handoffs should be written at the end of each session, even if work continues.
- If a session ends with work incomplete, the handoff must clearly state what remains.
