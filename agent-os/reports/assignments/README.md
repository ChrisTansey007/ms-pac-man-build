# Assignment Reports

> **Reports related to worker assignments, reassignments, and capability routing.**

## Purpose

This directory stores reports generated from assignment state, including:

- Assignment summaries (which worker is doing what).
- Capability coverage reports (are all needed capabilities covered?).
- Reassignment history.
- Worker utilization reports.

## Report Naming

```
assignment-report-[YYYY-MM-DD].md
reassignment-report-[YYYY-MM-DD].md
capability-coverage-[YYYY-MM-DD].md
```

## Generation

Assignment reports can be generated by:

- The Coordinator (Hermes) as part of queue audits.
- The human owner for project oversight.
- Automated scripts (future enhancement).
