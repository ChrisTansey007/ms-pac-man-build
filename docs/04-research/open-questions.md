# Open Questions

> **Track unresolved questions that need research or decision.**

## Purpose

When a worker encounters a question that cannot be answered with current information, it should be logged here. The human owner or a future worker can then address it.

## Format

```markdown
### [Q-XXX] [Question]

- **Raised by:** [Worker]
- **Date:** [YYYY-MM-DD]
- **Context:** [Why this matters]
- **Blocking:** [YES/NO — does this block progress?]
- **Status:** [open / researching / answered]
- **Answer:** [To be filled when resolved]
```
